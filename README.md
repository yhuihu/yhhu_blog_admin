## 简介

本博客是一个前后端分离的轻量级个人博客系统。

本后台管理系统使用了[vue-admin-template](https://github.com/PanJiaChen/vue-admin-template)作为基础模板

## 构建和运行

```bash
# 国内建议使用cnpm install
npm install
# run前可根据实际情况修改.env.development和.env.production中相关内容
npm run dev
# 打包
npm run build
```
